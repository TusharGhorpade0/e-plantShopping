# e-plantShopping 

An online plant shopping web application built using React and Redux Toolkit.

## Features

- Landing page with background image
- About Us section
- Get Started button
- Product listing page
- Multiple plant categories
- Add to Cart functionality
- Dynamic cart quantity updates
- Increment / Decrement quantity
- Delete items from cart
- Total cost calculation
- Checkout button

## Technologies Used

- React
- Redux Toolkit
- CSS
- JavaScript
